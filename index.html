ID card authentication examples

The following is a list of pages which provide examples of SSL authentication integration.

  • Require. This location is configured using the SSLVerifyClient require directive.
  • Optional. This location is configured using SSLVerifyClient optional directive.
  • Deny. This location is configured using the SSLVerifyDepth 0 directive. It will always deny connection. The important thing to note is the behaviour of the browser when the two locations above are accessed after accessing this one.
  • OCSP. This page provides an example of OCSP certificate verification query implemented at the application level.
  • LDAP. This is an example of LDAP-based (unofficial, but free) online certificate validation.
  • Login/Logout. This page shows examples of login/logout functinality.
